# PlanetaLibre

An Atom or RSS feeds aggregator for [Gemini](https://gemini.circumlunar.space/) written in `Lua`.

## Dependencies

* [LuaRocks](https://luarocks.org/)

## Installation

    git clone https://github.com/ricardogj08/planetalibre.git
    cd planetalibre
    sudo luarocks make
